# 考研数学二真题管理与AI答疑系统
## 项目简介
轻量化静态网页工具，面向考研数二备考学生，实现真题录入存储、题库浏览、豆包方舟AI智能答疑、LaTeX数学公式渲染功能，本地npx serve一键启动。

## 环境依赖
1. 本地安装Node.js环境，确保终端可以执行npx命令
2. 自行申请火山引擎豆包方舟API密钥、接口地址、模型标识

## 启动步骤
1. 克隆仓库或下载压缩包，解压得到项目文件夹【考研数二真题系统】
2. 打开终端（CMD/PowerShell/终端），cd切换到项目根目录
3. 执行启动命令：
```bash
npx serve .
## 核心技术栈
- 页面搭建：HTML + Bootstrap
- 数学公式渲染：KaTeX，兼容高数、线代标准LaTeX语法
- 本地持久化存储：浏览器localStorage（存储真题数据、AI接口配置）
- AI对话能力：火山引擎豆包方舟API，遵循OpenAI标准请求格式

## 常见问题解决方案
1. 点击按钮无响应：打开浏览器无痕/隐私窗口访问页面，使用Ctrl+F5强制清空缓存刷新
2. AI答疑没有返回结果：仔细核对API密钥、接口地址、模型名称，不要带多余空格
3. 数学公式显示错乱：题干、解答内容必须用 `$公式内容$` 标准LaTeX格式包裹
